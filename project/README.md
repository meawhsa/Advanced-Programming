# سیستم مدیریت دانشجویان و نمرات

## معرفی پروژه
این پروژه با زبان Python و بر اساس مفاهیم برنامه‌نویسی شی‌گرا (OOP) پیاده‌سازی شده است. هدف آن مدیریت اطلاعات دانشجویان، درس‌ها و نمرات و انجام تحلیل‌های آماری است.

## قابلیت‌ها
- افزودن و حذف دانشجو
- افزودن و حذف درس
- ثبت نمره برای هر دانشجو
- نمایش اطلاعات و کارنامه دانشجو
- محاسبه معدل (GPA)
- محاسبه مجموع واحدها
- مرتب‌سازی دانشجویان بر اساس:
  - معدل
  - تعداد واحد
  - نام
- تحلیل آماری با NumPy:
  - میانگین
  - بیشترین مقدار
  - کمترین مقدار
  - انحراف معیار
  - تعداد دانشجویان قبول و مردود

## ساختار کلاس‌ها
### Person
کلاس پایه شامل نام و متد نمایش اطلاعات.

### Student
ارث‌بری از Person و نگهداری نمرات، دروس، معدل و تعداد واحد.

### Lesson
نگهداری نام درس و تعداد واحد.

### EducationSystem
مدیریت کل سیستم آموزشی، دانشجویان، دروس و عملیات مختلف.

## مفاهیم استفاده‌شده
- Object-Oriented Programming (OOP)
- Inheritance
- Polymorphism
- Property
- Encapsulation
- Static Method
- Class Variable
- Dunder Method (`__str__`)

## الگوریتم مرتب‌سازی
در این پروژه از الگوریتم Merge Sort استفاده شده است.

**پیچیدگی زمانی:**
- Best Case: O(n log n)
- Average Case: O(n log n)
- Worst Case: O(n log n)
